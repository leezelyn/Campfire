Three.js で作られた、インタラクティブな夜の焚き火シーンです。  
プロジェクトは主に単一の `index.html` ファイルで構成されており、Three.js は CDN から読み込まれます。ビルドや依存関係のインストールは不要です。

---

## 概要

**Campfire** は、ブラウザ上で動作する夜の焚き火デモです。粒子による炎、火の粉、煙、月明かり、霧、天気調整、Web Audio によるプログラム生成サウンドを組み合わせ、静かな夜のキャンプの雰囲気を表現しています。

以下の学習や実験に役立ちます。

- Three.js による 3D シーン構築
- 炎、火の粉、煙の粒子表現
- Web Audio による音の生成
- 3D 空間音響と距離減衰
- 簡単な物理シミュレーション
- リアルタイムなパラメータ調整
- 多言語 UI

---

## 主な機能

### 夜の焚き火シーン

炎、薪、火の粉、煙、月明かり、霧、雲量、小雨、夜の環境光を組み合わせ、インタラクティブなキャンプ空間を作ります。

### リアルタイム調整パネル

画面右上のパネルから、以下の項目をリアルタイムで調整できます。

- 炎の強さ
- 熱による上昇力
- 火の粉の発生量
- 煙の量
- 風速
- 全体音量
- パチパチ音の頻度
- 距離減衰
- 空気による高音域の吸収
- 小雨
- 雲量
- 月相
- 月明かりの強さ
- 霧の濃さ
- 画面露出

すべての変更は即座に反映されます。

### 薪をくべる操作

下部の **薪をくべる** ボタンを押すと、薪がカメラ側から放物線を描いて焚き火に投げ込まれます。

この操作では以下の表現が行われます。

- 放物線運動
- 回転アニメーション
- 火の粉の発生
- 煙の発生
- 3D 位置に基づく着地音
- 燃料の増加
- 炎の強化
- 一時的なパチパチ音の増加

燃料は時間とともに燃焼し、炎の勢いも徐々に落ち着いていきます。

### 焚き火インタラクション

以下の 4 種類のインタラクションがあります。

- **マシュマロを焼く**：マシュマロを火に近づけ、徐々に焦げ目をつけます。
- **お湯を沸かす**：ケトルを炎の上に吊るし、加熱後に蒸気を発生させます。
- **たいまつに火をつける**：たいまつを炎に近づけて点火し、もう一度押すと消火できます。
- **松ぼっくりを投げる**：松ぼっくりを火に投げ入れ、火の粉を散らして少量の燃料を追加します。

### 三言語対応 UI

UI は以下の三言語に対応しています。

- 中文
- English
- 日本語

ボタン、状態表示、進行状況などが即時に切り替わります。

### プログラム生成サウンド

すべての音は Web Audio API によってリアルタイム生成されます。外部音声ファイルは不要です。

生成される音の例：

- 炎の低い燃焼音
- 薪のパチパチ音
- ときどき発生する破裂音
- 薪が落ちる鈍い音
- 距離による音量減衰
- 遠くなるほど高音が弱くなる空気吸収表現

## 実行方法

このプロジェクトは ES Module を使用しているため、ローカル HTTP サーバー経由で開くことを推奨します。

```bash
python -m http.server 8341
```

その後、ブラウザで以下にアクセスします。

```text
http://localhost:8341
```

ページを開いた後、任意の場所をクリックすると音声が有効になります。  
これは、多くのブラウザが Web Audio の再生にユーザー操作を必要とするためです。

---

## 操作方法

| 操作 | 説明 |
|---|---|
| マウスドラッグ | カメラを回転 |
| マウスホイール | ズームイン / ズームアウト |
| ページをクリック | 音声を開始 |
| 「薪をくべる」をクリック | 焚き火に薪を追加 |
| 各インタラクションボタンをクリック | マシュマロ、湯沸かし、たいまつ、松ぼっくりを操作 |
| パラメータパネルを調整 | 炎、音、天気、環境効果をリアルタイム変更 |

## 物理・演出シミュレーション

| 現象 | 実装 |
|---|---|
| 距離による音の減衰 | `PositionalAudio` と inverse 距離モデル |
| 両耳定位 | HRTF 空間音響 |
| 空気吸収 | 距離が遠いほど高音域を弱める |
| 火の光の減衰 | 点光源の平方反比減衰 |
| 炎の上昇 | 熱上昇力、空気抵抗、乱流、中心方向への力 |
| 焔の層構造 | 青い焔底、白黄色の内焔、橙赤色の外焔 |
| 火の粉の動き | 重力、空気抵抗、熱による上昇流 |
| 煙の拡散 | 上昇、膨張、風による流れ、フェードアウト |
| パチパチ音 | ランダムに発生する短い破裂音 |
| 燃焼音 | 炎の強さと連動する低周波ノイズ |
